欢迎
=============

   Dice!是一款基于酷Q,采用 **C++** 编写的在线掷骰插件,编译环境为 **VS2017** 。

   如果您不了解酷Q是什么，请访问:`酷Q官网 <http://cqp.cc/>`_  

   如果您想要查看源代码:请访问:`Github <http://github.com/w4123/dice-of-coc/>`_  

   如果您想要配置机器人:请自行使用源代码编译或者加入QQ群 **624807593** 来下载插件,然后参考 `此链接 <./Configure.html>`_

   如果您想要查看机器人的使用方法: `请点击此链接 <./Usage.html>`_
